Agrupamento de Seletores

Podemos aplicar as mesmas regras do CSS para seletores diferentes. Para isso podemos separar os seletores usando uma virgula(,)

Assim não precisamos ficar compiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!

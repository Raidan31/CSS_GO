HSL e HSLA

HSL (Hue, Saturation e lightness)

HSLA (Hue, Saturation, lightness e Alpha)

Existem varias formas de definirmos cores utilizando o CSS

Por meio do site - w3schools, é possivel conferir uma lista de cores aceitas no CSS

  • HUE (tom) É o grau na roda de cores.
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation(Saturação) Valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Lightness(Luminosidade) Luminosidade da cor
    • 0% é preto
    • 100% é branco
  • Alpha Transparencia
    • De 0.0 há 1.0

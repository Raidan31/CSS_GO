Seletores CSS

Bem Vindo

No CSS nos possuimos os seletores por ID

  • Seletor de tag tipo:Busca elementos por tag html especifica
  • Seletor de ID(#):Busca elementos através do atributo "ID"
  • Seletor de Classes(.):Busca elementos através do atributo "class" diferente do id, é possivel colocar a mesma classe em multiplos elementos
  • Seletor Universal(*):Seleciona todos os elementos html.
  • Seletor de Atributo([atrib]):Seleciona elementos que possuam um atributo.

Redimensionando Imagens de Fundo:

Por meio do background-size é possivel redimensionar imagens de fundo

Auto é o valor padrão do background-size (conforme imagem abaixo), caso a imagem seja menor ela irá se replicar para caber no espaço:

Cover vai redimensionar para o tamanho selecionado, contudo irá cortar a imagem para caber caso seja necessário:

Contain vai redimensionar a imagem para que ela caiba completamente, mesmo que sobre espaço

Valor é possivel definir diretamente com um unico valor(largura) na propriedade background size em px ou %. Altura será automatica.

Valor1 e Valor2 é possivel definir diretamente com dois valores(largura x altura) na propriedade background size em px ou %
